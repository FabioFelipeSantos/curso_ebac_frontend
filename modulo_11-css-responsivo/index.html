Módulo de Aprendizado

Este projeto se destina à avaliação dos conhecimentos adquiridos no módulo de aprendizado Fundamentos do CSS Responsivo.

Uma das maiores qualidade de um excelente web site é a sua aptidão à se adaptar aos diferentes tamanhos de tela em que será apresentado. Nesse ponto, o CSS possui uma série de ferramentas diferentes para lidar com a variação do tamanho da tela. Se destacam a possibilidade de: setar tamanhos em função do comprimento e altura da tela, como as unidades vw e vh; setar valores para as propriedades de forma dinâmica, usando a função clamp(); adicionar breakpoints dependentes do tamanho da tela com a regra atribuída (at-rule) @media.

O web developer que venha dominar tais técnicas se torna um melhor profissional, mais capacitado a lidar com as diferentes formas de apresentar os conteúdos desejados em web sites.

O Projeto

Se trata da criação de uma web store, uma loja de vendas. Deveria-se criar um layout responsivo a telas de celulares, tablets e desktops. A utilização adequada das funcionalidades do CSS deveria ser aplicada levando em consideração a aplicação dos conceitos de implementação do Mobile First, em que primeiro devemos implementar o website para telas de mobiles, como smartphones, e após isso adicionar a media queries com o uso da regra atribuída @media.

Funcionalidades exigidas e implementadas:

  1. Criar um layout responsivo de uma loja virtual;
  2. Criar uma seção “sobre a loja”, nesta seção teremos duas colunas uma com a imagem da loja e outra com um texto;
  3. Criar uma seção de produtos onde haverá uma listagem de 4 produtos, lado a lado, com a imagem, nome e descrição do produto e um botão para adicionar ao carrinho;
  4. Seguir as adaptações de layout propostas e fazer a publicação na Vercel nos enviando link do projeto.

Funcionalidades adicionais implementadas:

  1. Um footer para informações sobre a implementação;
  2. Um menu e um submenu;
  3. Abertura do submenu com o click sobre o link Produtos;
  4. Scroll da página do menu até a seção correspondente;
  5. Criação desse modal e ajustes no scroll da página quando o modal estiver ativo

Tecnologias utilizadas

  • HTML5
  • CSS3
  • Vanilla JS (ES6)
  • jQuery v3.7.1
  • VS Code

Desafios encontrados:

  • Mudar a forma de implementação para se adequar ao paradigma Mobile First. Resolvido com CSS.
  • Alguns ajustes do scroll do menu até o item correspondente. Resolvido com jQuery e CSS.
  • Criação de um menu escondido e apresentado quando um evento é disparado no parente mais próximo do elemento. Resolvido com o jQuery e com CSS.

Conclusão

Utilizando de ferramentas básicas de CSS somos capazes de criar com qualidade websites de forma responsiva que se adéqua a qualquer tipo de tela. A utilização da regra atribuída @media torna o código menos dependente da quantidade de comandos em JS que se deve executar para adaptar um site via código. Apenas a estilização é capaz de responder à essa tarefa.
