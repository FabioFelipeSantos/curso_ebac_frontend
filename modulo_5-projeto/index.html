Fábio Santos

Matemático. Professor. Pesquisador. Desenvolvedor. Programador. Eterno Estudante...

Formado em matemática, atuo como professor desde 2008. O que me guia é a busca por ensinar o indivíduo pensante a querer e buscar ser MAIS inteligente a cada dia, e não regras sem sentido para solucionar equações sem significado. Espero alcançar mentes brilhantes que tenham vontade de mudar seu universo por meio do aprendizado de exatas. Em 2022 comecei alguns cursos no Youtube no meu canal Derivada de Konoha. No canal, meu foco é ensinar coisas que não tive a oportunidade de ensinar como matemático. Em 2024 tomei a decisão de construir mais uma habilidade de programação, a de Web Developer, focado em Backend. O intuito é mudar de carreira e também construir minha própria plataforma de ensino.

Pesquisador

2017 - Atual

Desde antes do meu mestrado, porém com maior intensidade durante e após ele, eu faço pesquisas na área da otimização matemática e computacional, tendo trabalhado com Algoritmos Heurísticos para otimização de funções não convexas. A principal aplicação da pesquisa foi com foco em Otimização Estrutural e Análise Computacional de Estruturas, utilizando o Método dos Elementos Finitos como procedimento de aproximação. Trabalhei também desenvolvendo aplicações para análise de problemas Estocásticos em Dinâmica Estrutural. Outro ponto de pesquisa foi em Redução Modal para Elementos Finitos.

Programas Implementados

Análise de Estruturas pelo método da Rigidez - Matlab Programa que utiliza o Método da Rigidez para análise de estruturas em 2d como Treliças, Vigas e Pórticos
Análise Linear de Estruturas pelo Método dos Elementos Finitos - Matlab Programa utilizando o Método dos Elementos Finitos para análise linear de estruturas como barras, vigas e placas em 2d.
Otimização Global utilizando o Efficient Global Optimization - EGO - Matlab Implementação do algoritmo Efficient Global Optimization - EGO para otimização de funções multimodais (não-convexas). Utiliza o Kriging Determinístico como radial basis function para criação dos metamodelos de substituição das funções objetivo.
Otimização Estocástica utilizando sEGO - Stochastic Efficient Global Optimization - Matlab Implementação do algoritmo Efficient Global Optimization - EGO para otimização de funções multimodais (não-convexas). Utiliza o Kriging Estocástico como radial basis function para criação dos metamodelos de substituição das funções objetivo que são sujeitas a erros ou aleatoriedades em suas propriedades.
